# 농장 운영 방법
1. Organization에 본인의 이름으로 Repository를 생성한다.
2. Readme.md 파일에 본인의 농장 이름을 적는다.
3. 주간목표를 적고 해당하는 주의 기간을 적는다.
4. 휴식일을 지정한다.
5. 주간목표를 추가한다.
6. 주간목표에 잔디를 심은 저장소의 위치링크를 첨부한다.
